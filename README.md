# Expense Tracker

**Автор:** [Лиза] [Соколова]
**Язык:** Python 3.x

## Описание программы

Это графическое приложение для учета личных финансов. Оно позволяет фиксировать ежедневные расходы по категориям (еда, транспорт и т.д.), анализировать траты за выбранные периоды и хранить историю в файле формата JSON.

## Технологии

*   **Python:** Язык программирования.
*   **Tkinter:** Стандартная библиотека для создания графического интерфейса (*GUI*).
*   **JSON:** Формат хранения данных.
*   **Git:** Система контроля версий для отслеживания изменений в коде.

## Как использовать

1.  **Установка:**
    Убедитесь, что у вас установлен Python версии 3.6 или выше. Библиотека `tkinter` обычно входит в стандартный комплект поставки Python.
2.  **Запуск:**
    Откройте терминал в папке проекта и выполните команду:
    ```bash
    python expense_tracker.py
    ```
3.  **Интерфейс:**
    *   **Добавление расхода:** Введите сумму (число), категорию (например: *Продукты*) и дату в формате *ГГГГ-ММ-ДД* (например: *2026-05-02*). Нажмите кнопку «Добавить расход».
    *   **Фильтрация:** Введите название категории или период дат (начало и конец) и нажмите «Применить фильтр». Таблица обновится.
4.  **Автоматическое сохранение:** Все введенные данные сохраняются в файл `data.json` автоматически после каждого добавления или изменения.
